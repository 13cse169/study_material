1. Arrays / Lists

An array stores multiple values in a single variable. In Python arrays are handled using lists.

Example

arr = [10,20,30,40] print(arr[0]) print(arr[-1])
Practice Problem
Find the second largest element in a list.

arr=[10,20,4,45,99] arr.sort() print(arr[-2])

2. Strings

A string is a sequence of characters. It is widely used in search, pattern matching, and validation tasks.

Example

text = "python" print(text[::-1])
Practice Problem
Check if a string is a palindrome.

s="madam" if s==s[::-1]: print("Palindrome") else: print("Not Palindrome")

3. Recursion

Recursion happens when a function calls itself to solve a smaller part of the same task.

Example

def factorial(n): if n==1: return 1 return n * factorial(n-1) print(factorial(5))
Practice Problem
Write a recursive Fibonacci program.
def fib(n): if n<=1: return n return fib(n-1)+fib(n-2) print(fib(6))

4. Stack

Stack follows Last In First Out (LIFO).

stack=[] stack.append(10) stack.append(20) print(stack.pop())
Practice Problem
Check valid parentheses.
def valid(s): stack=[] for c in s: if c=="(": stack.append(c) elif c==")": if not stack: return False stack.pop() return not stack print(valid("()()"))

5. Queue

Queue follows First In First Out (FIFO).

from collections import deque q=deque() q.append(10) q.append(20) print(q.popleft())

6. Linked List

A linked list is a chain of nodes where each node stores data and a pointer to the next node.

7. Trees

A tree is a hierarchical data structure used in databases, file systems, and AI search algorithms.
