Introduction to Machine Learning

What is Machine Learning?
Machine Learning is a subset of Artificial Intelligence that enables computers to learn from data and improve their performance on tasks without being explicitly programmed. Instead of writing rules manually, we let the algorithm discover patterns.
Fig 1 — AI ⊃ Machine Learning ⊃ Deep Learning hierarchy
💡 Key Insight: Traditional programming = Data + Rules → Output. Machine Learning = Data + Output → Rules (the model learns the rules).
⚙️ Traditional Programming
Rules + Data → Output

You explicitly write IF-ELSE logic. The programmer defines every rule.
🧠 Machine Learning
Data + Output → Rules

You feed examples. The algorithm discovers the underlying rules itself.

The ML Workflow — Step by Step

Fig 3 — End-to-end ML Pipeline
1

Data Collection

Gather raw data from databases, APIs, web scraping, or sensors. Data quality determines model quality — "garbage in, garbage out."

2

Data Preprocessing

Handle missing values, remove duplicates, fix inconsistencies, normalize/scale numerical features, encode categorical variables.

3

Feature Engineering

Select, transform, or create new features from raw data that improve the model's ability to detect patterns.

4

Model Selection & Training

Choose an algorithm, split data into train/test sets, fit the model on training data to learn parameters.

5

Model Evaluation

Measure performance using metrics (Accuracy, F1, RMSE). Use cross-validation to ensure the model generalises well.

6

Deployment & Monitoring

Serve the model via API or embedded system. Continuously monitor performance as real-world data drifts over time.

Linear Regression

Definition
Linear Regression models the relationship between a dependent variable y and one or more independent variables x by fitting a straight line that minimises the sum of squared errors.
y = β₀ + β₁x₁ + β₂x₂ + … + βₙxₙ + ε
Fig 4 — Linear Regression: best-fit line minimising residuals
Cost Function (MSE)
Measures how wrong the model is. We minimise this during training using Gradient Descent.
MSE = (1/n) Σ(yᵢ − ŷᵢ)²
Gradient Descent
Iteratively adjusts weights in the direction that reduces the cost function.
θ := θ − α · ∂J/∂θ

Neural Networks

Artificial Neural Network (ANN)
Inspired by the human brain, an ANN is composed of layers of interconnected nodes (neurons). Each connection has a weight. The network learns by adjusting these weights through backpropagation.
Fig 6 — Feedforward Neural Network (3→4→4→2 architecture)
Activation Functions
Introduce non-linearity so the network can learn complex patterns. Without them, stacked layers collapse into a single linear transformation.
ReLU: f(x) = max(0, x)  |  Sigmoid: f(x) = 1/(1+e⁻ˣ)  |  Tanh: f(x) = (eˣ−e⁻ˣ)/(eˣ+e⁻ˣ)

Model Evaluation Metrics

Fig 7 — Confusion Matrix: foundation of classification metrics
Accuracy
Percentage of all correct predictions. Misleading on imbalanced datasets.
(TP + TN) / (TP + TN + FP + FN)
Precision
Of all positive predictions, how many were actually positive? Minimises false alarms.
TP / (TP + FP)
Recall (Sensitivity)
Of all actual positives, how many did we catch? Critical in medical diagnoses.
TP / (TP + FN)
F1 Score
Harmonic mean of Precision and Recall. Best single metric for imbalanced classes.
2 × (P × R) / (P + R)
Random Forest96%

09

Deep Learning

Deep Learning
A subset of ML using deep neural networks (many hidden layers). Automatically learns hierarchical feature representations from raw data — no manual feature engineering required for images, text, or audio.
Fig 9 — CNN: automatically learns edges → shapes → objects in hierarchical layers
🖼️ CNN — Convolutional Neural Net
Best for image data. Uses convolution filters to detect local spatial features like edges, textures, and patterns.
🔄 RNN / LSTM
Best for sequential data (text, time series). Maintains a hidden state across time steps to capture temporal dependencies.
🤖 Transformer
Attention-based architecture powering GPT, BERT. Processes entire sequences in parallel and captures long-range dependencies efficiently.
🎨 GAN
Two networks (Generator vs Discriminator) compete in a game — Generator creates fake data, Discriminator tries to spot it. Used for image generation.
💡 Key Takeaway: The more data you have, the more a deep learning model tends to outperform classical ML. For small datasets, classical algorithms (Random Forest, SVM, XGBoost) often win on both speed and accuracy.
