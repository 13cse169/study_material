ASR · TTS · Speaker Recognition
🔹Automatic Speech Recognition

End-to-end ASR maps raw audio waveforms to text. Modern systems use transformers pre-trained on 680K hours (Whisper) achieving human-level transcription.

Python · Whisper ASR
import whisper

model = whisper.load_model('large-v3')
result = model.transcribe('audio.mp3')
print(result['text'])

# HuggingFace pipeline
from transformers import pipeline
asr = pipeline('automatic-speech-recognition',
              model='openai/whisper-large-v3')
print(asr('audio.mp3'))

Q01Explain CLIP and how it enables zero-shot image classification.
hard+
CLIP (Contrastive Language-Image Pre-training, OpenAI 2021) trains dual encoders — one for images (ViT), one for text (Transformer) — with a contrastive loss: pull matching (image, text) pairs together, push non-matching apart. Trained on 400M internet image-caption pairs. Zero-shot classification: encode all class names as text ("a photo of a cat"), encode the query image, find the most similar text embedding by cosine similarity. Achieves 76% ImageNet top-1 with no ImageNet training. Enables: zero-shot detection (OWL-ViT), image-text retrieval, text-guided generation (Stable Diffusion uses CLIP text encoder).
Q02How does object detection differ from image classification?
mid+
Classification: Input image → single class label. No location information. Detection: Input image → list of (bounding box, class, confidence) tuples. Two paradigms: Two-stage (R-CNN family): Region proposal network generates candidate boxes → RoI pooling → classification. High accuracy, slower. One-stage (YOLO, SSD): Directly predict boxes and classes at each grid cell. Faster, slightly lower accuracy. Transformer-based (DETR): End-to-end, no anchor boxes. Slow training but elegant. Modern: YOLOv8/YOLOv10 for real-time, DETR variants for accuracy.
