TEXT CLASSIFICATION

Sentiment · Topic · Intent with Deep Learning
Python · BERT Fine-tune for Classification
from transformers import AutoTokenizer, AutoModelForSequenceClassification, Trainer, TrainingArguments
from datasets import load_dataset

ds = load_dataset('imdb')
tok = AutoTokenizer.from_pretrained('distilbert-base-uncased')

def tokenize(b): return tok(b['text'], truncation=True, padding=True)
ds = ds.map(tokenize, batched=True)

model = AutoModelForSequenceClassification.from_pretrained(
    'distilbert-base-uncased', num_labels=2)

args = TrainingArguments(output_dir='./out', num_train_epochs=3,
    per_device_train_batch_size=32, learning_rate=2e-5, fp16=True)
Trainer(model=model, args=args,
    train_dataset=ds['train'], eval_dataset=ds['test']).train()

Q01How does Word2Vec's skip-gram work?
mid+
Skip-gram trains a shallow neural network to predict surrounding context words given a centre word. For pair (centre, context): maximise P(context|centre) = softmax(Wout · embed_centre). Negative sampling approximates this: for each positive pair, sample k negatives and push them apart. The word vectors are the input embedding matrix Win. Trained on massive corpora, these vectors encode syntactic and semantic relationships. Skip-gram works better for rare words; CBOW is faster for frequent words.
Q02What is the difference between extractive and abstractive summarisation?
mid+
Extractive: Select and concatenate sentences from source. Approaches: TF-IDF ranking, TextRank (graph-based), BERT-based sentence scoring. Always grammatical, can't rephrase. Abstractive: Generate new text — may compress, rephrase, merge sentences. Requires seq2seq models (T5, BART, PEGASUS). Better conciseness and coherence but can hallucinate. PEGASUS (pre-trained with gap-sentence generation) achieves SOTA on CNN/DailyMail. In production: extractive for factual domains (legal, medical); abstractive for user-facing summaries.
Advanced
Q03Explain attention mechanism in seq2seq models.
hard+
Bahdanau attention (2015) solved the encoder bottleneck. Instead of compressing source into one vector, the decoder at each step attends to all encoder states: score(sₜ,hᵢ) = vᵀ·tanh(Ws·sₜ+Wh·hᵢ), α = softmax(scores), context cₜ = Σ αᵢhᵢ. This lets the decoder focus on relevant source positions. Extended in Transformers to self-attention (each position attends to all others in same sequence) and multi-head attention (parallel attention patterns). The attention weights also provide interpretability — you can visualise which source words the decoder attends to.
