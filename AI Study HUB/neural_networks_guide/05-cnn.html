CONVOLUTION OPERATION

How CNNs process spatial data
🔹Why Convolutions?

A fully-connected layer on a 224×224×3 image would need 224×224×3×hidden_size weights — millions per layer, no spatial awareness. Convolutions share weights spatially — a 3×3 filter applied everywhere: efficient, translation-invariant, captures local patterns.

Output[i,j] = Σₖ Σₗ Input[i+k, j+l] · Filter[k,l] + bias Output_size = (Input_size - Kernel + 2·Padding) / Stride + 1
Python · Conv2d Parameters
import torch.nn as nn

# Conv2d(in_channels, out_channels, kernel_size, stride, padding)
conv = nn.Conv2d(3, 64, kernel_size=3, stride=1, padding=1)
# Input: (B, 3, H, W) → Output: (B, 64, H, W)  (same spatial dims with pad=1)

# Parameter count: 3×64×3×3 + 64 = 1,792
print(sum(p.numel() for p in conv.parameters()))  # 1792

Python · ResNet Block
class ResBlock(nn.Module):
    def __init__(self, channels):
        super().__init__()
        self.block = nn.Sequential(
            nn.Conv2d(channels, channels, 3, padding=1, bias=False),
            nn.BatchNorm2d(channels), nn.ReLU(),
            nn.Conv2d(channels, channels, 3, padding=1, bias=False),
            nn.BatchNorm2d(channels))
        self.relu = nn.ReLU()
    def forward(self, x):
        return self.relu(self.block(x) + x)  # skip connection!

TRANSFER LEARNING

Fine-tune pretrained models on custom data
Python · Transfer Learning
import torchvision.models as models

# Load pretrained ResNet50
model = models.resnet50(weights='IMAGENET1K_V2')

# Freeze all layers
for param in model.parameters():
    param.requires_grad = False

# Replace final layer for your task
model.fc = nn.Sequential(
    nn.Dropout(0.5),
    nn.Linear(2048, num_classes))

# Phase 2: unfreeze and fine-tune with small LR
for param in model.layer4.parameters():
    param.requires_grad = True

optimizer = torch.optim.AdamW([
    {'params': model.fc.parameters(), 'lr': 1e-3},
    {'params': model.layer4.parameters(), 'lr': 1e-4}])
✅Rule: For small custom datasets (<10K images), freeze everything and train only the head. For medium datasets, fine-tune last 2 blocks. For large datasets, fine-tune everything with small LR.

INTERVIEW Q&A

CNN · 8 Questions
Intermediate
Q01Why use 3×3 convolutions instead of larger kernels?
mid+
VGG demonstrated that two 3×3 convs have the same receptive field as one 5×5 but with fewer parameters (2×9×C² vs 25×C²) and an extra non-linearity. Three 3×3 convs ≈ one 7×7 with 3×9=27 vs 49 params. Benefits: fewer parameters → less overfitting, more efficient computation, more non-linearities → more expressive. Modern architectures (ResNet, EfficientNet) use 3×3 almost exclusively, with 1×1 convolutions for channel mixing and dimension reduction.
Q02What is the purpose of pooling layers?
mid+
Pooling reduces spatial dimensions, providing: (1) Translation invariance — a shifted feature still activates the same pool region; (2) Dimension reduction — less computation in subsequent layers; (3) Increasing receptive field — deeper layers see larger image regions. Max pooling takes the maximum value in a window — detects whether a feature is present anywhere in the region. Average pooling computes the mean — smoother, often used at the end (global average pooling replaces flatten). Trend: modern architectures use strided convolutions instead of explicit pooling — learnable downsampling.
Advanced
Q03What is the residual connection and why does it solve the degradation problem?
hard+
Before ResNet (He 2015), adding more layers made networks worse — not due to overfitting but because deep networks were harder to optimise. The degradation problem: the network should at minimum be able to learn identity mappings, but gradient descent struggles to find them. Residual connections make identity the default: y = F(x) + x. Now the network only needs to learn the residual F(x) = desired_output - x. If F(x)=0, we get identity for free. Crucially, the skip connection provides a gradient highway: ∂L/∂x = ∂L/∂y · (∂F/∂x + 1). The '+1' ensures the gradient reaches early layers. This enabled training networks with 100-1000+ layers.
