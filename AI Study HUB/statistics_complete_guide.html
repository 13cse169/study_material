Part 1

Beginner Statistics

1. Introduction to Statistics

Statistics is the science of collecting, organizing, analyzing, interpreting, and presenting data. It helps us make sense of numerical information and draw conclusions from data.

1.1 Types of Statistics

Descriptive Statistics

Describes and summarizes data using measures like mean, median, mode, and standard deviation. It does NOT draw conclusions beyond the data.

  • Example: Average score of students in a class
  • Example: Most common shoe size sold in a store

Inferential Statistics

Uses sample data to make inferences or predictions about a larger population. It involves probability and hypothesis testing.

  • Example: Predicting election results from a poll
  • Example: Testing if a new drug works better than existing ones

1.2 Types of Data

Qualitative (Categorical) Data

  • Nominal: No order — colors, gender, country
  • Ordinal: Has order but no fixed interval — ratings: good, better, best

Quantitative (Numerical) Data

  • Discrete: Countable values — number of children, number of cars
  • Continuous: Measurable values — height, weight, temperature

1.3 Measures of Central Tendency

These measures describe the center or typical value of a dataset.

Mean (Average)

Mean (x̅) = Σx / n    where n = number of values

Example: {4, 7, 13, 16} → Mean = (4+7+13+16)/4 = 10

Median

The middle value when data is sorted. If two middle values exist, take their average.

{3, 5, 7, 9, 11} → Median = 7
{3, 5, 7, 9} → Median = (5+7)/2 = 6

Mode

The value that appears most frequently. Can be unimodal, multimodal, or have no mode.

{2, 3, 3, 5, 7, 7, 7} → Mode = 7

1.4 Measures of Spread (Variability)

Range

Range = Maximum value − Minimum value

Variance

Population Variance (σ²) = Σ(x − μ)² / N
Sample Variance (s²) = Σ(x − x̅)² / (n − 1)

Standard Deviation

The square root of variance — measures how spread out values are from the mean.

Population SD (σ) = √[Σ(x − μ)² / N]
Sample SD (s) = √[Σ(x − x̅)² / (n−1)]

Interquartile Range (IQR)

IQR = Q3 − Q1    (Q1 = 25th percentile, Q3 = 75th percentile)
Beginner Interview Questions
Q What is the difference between mean, median, and mode? +
A Mean is the arithmetic average (sum divided by count). Median is the middle value when data is sorted. Mode is the most frequently occurring value. Mean is sensitive to outliers; median is more robust. Example: {1, 2, 3, 100} — Mean=26.5, Median=2.5, Mode=none.
Q What is the difference between descriptive and inferential statistics? +
A Descriptive statistics summarizes and describes the features of a dataset (e.g., mean, standard deviation of a class's test scores). Inferential statistics uses sample data to make predictions or draw conclusions about a larger population (e.g., using poll data to predict election outcomes).
Q What is variance and why do we use n−1 in sample variance? +
A Variance measures how spread out data points are from the mean. We use n−1 (Bessel's correction) to correct for bias — dividing by n underestimates the population variance because a sample tends to cluster closer to its own mean than to the true population mean.
Q What are outliers and how do they affect statistics? +
A Outliers are data points that differ significantly from other observations. They heavily influence the mean but have less effect on the median and mode. In {2, 3, 4, 5, 100}, the mean is 22.8 (unrepresentative), while the median (4) is more representative.

Intermediate Statistics

2. Probability

Probability measures the likelihood of an event occurring, expressed as a value between 0 (impossible) and 1 (certain).

P(A) = Number of favorable outcomes / Total number of outcomes

2.1 Rules of Probability

Addition Rule

P(A ∪ B) = P(A) + P(B) − P(A ∩ B)
For mutually exclusive events: P(A or B) = P(A) + P(B)

Multiplication Rule

P(A ∩ B) = P(A) × P(B|A)   [Dependent]
P(A ∩ B) = P(A) × P(B)     [Independent]

Conditional Probability

P(A|B) = P(A ∩ B) / P(B)
Probability of A given that B has already occurred.

2.2 Bayes' Theorem

Allows us to update the probability of an event based on new evidence.

P(A|B) = [P(B|A) × P(A)] / P(B)
Classic Example: A medical test is 99% accurate. If only 1% of people have a disease, what is the true probability a positive test result means you have the disease? Bayes' theorem reveals it may still be quite low — this is the "base rate fallacy."

2.3 Probability Distributions

Binomial Distribution

Models the number of successes in n independent trials, each with probability p.

P(X=k) = C(n,k) × p^k × (1−p)^(n−k)

Mean = np    |    Variance = np(1−p)
  • Example: Probability of getting exactly 3 heads in 10 coin flips

Poisson Distribution

Models events occurring in a fixed interval when events happen at a known average rate λ.

P(X=k) = (e^(−λ) × λ^k) / k!    where λ = average rate

Mean = Variance = λ
  • Example: Number of emails received per hour

Normal Distribution

A symmetric, bell-shaped distribution described by mean (μ) and standard deviation (σ).

f(x) = (1/σ√2π) × e^[−(x−μ)²/(2σ²)]

68% within 1σ  |  95% within 2σ  |  99.7% within 3σ   (Empirical Rule)

2.4 Z-Score & Central Limit Theorem

Z-Score

Z = (X − μ) / σ
How many standard deviations a value is from the mean.

Central Limit Theorem (CLT)

The sampling distribution of the sample mean approaches normality as n increases (n ≥ 30), regardless of the population's distribution.

Standard Error (SE) = σ / √n
⭐ Why it matters: CLT is the foundation of confidence intervals, hypothesis testing, and most inferential statistics.
Intermediate Interview Questions
Q Explain Bayes' Theorem with a real-world example. +
A Bayes' theorem: P(A|B) = P(B|A)×P(A)/P(B). Example — Spam filters: If P(spam)=0.3, P('buy' in spam)=0.8, P('buy' in email)=0.35, then P(spam|'buy') = (0.8×0.3)/0.35 = 0.686. So 68.6% chance the email is spam. This is exactly how Naive Bayes classifiers work.
Q What is the Central Limit Theorem and why is it important? +
A CLT states that sample means approach a normal distribution as sample size increases (n≥30), regardless of the population distribution. It's important because: (1) enables normal-distribution-based tests even for non-normal populations, (2) foundation for confidence intervals and hypothesis testing, (3) explains why real-world measurements often follow a normal distribution.
Q What is the difference between a Z-test and a T-test? +
A Z-test: used when population SD is known and/or n>30. T-test: used when population SD is unknown and n<30. The t-distribution has heavier tails to account for extra uncertainty from estimating σ from data. As n increases, the t-distribution converges to the normal distribution.
Q What is the difference between Normal, Binomial, and Poisson distributions? +
A Normal: continuous, symmetric bell curve (height, weight). Binomial: discrete, counts successes in n independent yes/no trials with probability p (coin flips). Poisson: discrete, counts events in fixed time/space with known average rate λ (calls per hour). Poisson is a limit of Binomial as n→∞ and p→0 with np=λ constant.

Advanced Statistics

3. Hypothesis Testing

A formal procedure for using data to make decisions about population parameters.

3.1 Key Concepts

Null and Alternative Hypotheses

  • H₀ (Null): Assumes no effect, no difference, or status quo
  • H₁ or Hₐ (Alternative): What we want to prove

Type I and Type II Errors

DecisionH₀ TrueH₀ False
Reject H₀ ⚠️ Type I Error (α) — False Positive ✅ Correct (Power = 1−β)
Fail to Reject H₀ ✅ Correct Decision ⚠️ Type II Error (β) — False Negative

P-Value

Probability of observing results as extreme as the actual results, assuming H₀ is true.

  • p < α (usually 0.05): Reject H₀
  • p ≥ α: Fail to reject H₀

3.2 Common Statistical Tests

T-Tests

One-Sample:
t = (x̅ − μ₀) / (s / √n)
df = n − 1
Two-Sample:
t = (x̅₁ − x̅₂) / √(s²₁/n₁ + s²₂/n₂)

Chi-Square Test (χ²)

χ² = Σ[(O − E)² / E]    where O=Observed, E=Expected
Tests association between categorical variables or goodness of fit.

ANOVA (Analysis of Variance)

F = (Between-group variance) / (Within-group variance) = MSB / MSW
  • One-way ANOVA: One independent variable
  • Two-way ANOVA: Two independent variables (with interaction effects)
  • Assumptions: Normality, Homogeneity of variance, Independence

3.3 Confidence Intervals

CI = x̅ ± (Z* or t*) × (s / √n)

A 95% CI means: if we repeated the sampling process many times, 95% of constructed intervals would contain the true population parameter.

4. Regression Analysis

4.1 Simple Linear Regression

Y = β₀ + β₁X + ε

β₀ = intercept (Y when X=0)  |  β₁ = slope  |  ε = error term

Least Squares Estimation

β₁ = Σ[(xᵢ−x̅)(yᵢ−ȳ)] / Σ(xᵢ−x̅)²
β₀ = ȳ − β₁ × x̅

R² (Coefficient of Determination)

R² = 1 − (SSR / SST) = Explained Variance / Total Variance
R²=0.85 means the model explains 85% of variance in Y.

4.2 Multiple Linear Regression

Y = β₀ + β₁X₁ + β₂X₂ + … + βₖXₖ + ε
  • Adjusted R² penalizes for unnecessary predictors
  • Multicollinearity: check with VIF (Variance Inflation Factor)
  • Assumptions: Linearity, Normality of residuals, Homoscedasticity, No multicollinearity

4.3 Logistic Regression

Used when the dependent variable is binary (0 or 1). Models the probability of an outcome.

log(p/(1−p)) = β₀ + β₁X₁ + … + βₖXₖ

P(Y=1) = 1 / (1 + e^(−(β₀ + β₁X)))
  • Output is probability (0 to 1), not a continuous value
  • Uses Maximum Likelihood Estimation (MLE), not least squares
  • Evaluate with: AUC-ROC, Confusion Matrix, Log Loss

5. Advanced Topics

5.1 Correlation vs Causation

Pearson r = Σ[(xᵢ−x̅)(yᵢ−ȳ)] / √[Σ(xᵢ−x̅)² × Σ(yᵢ−ȳ)²]

r=+1: Perfect positive  |  r=−1: Perfect negative  |  r=0: No linear correlation
⚠️ Important: Correlation does NOT imply causation. Ice cream sales correlate with drowning rates — both are caused by hot weather.

5.2 Bootstrap & Resampling

A non-parametric technique that estimates the sampling distribution by resampling with replacement from the original data.

  • Useful when theoretical distributions are unknown
  • Can estimate standard errors and confidence intervals
  • Used in ML ensemble methods: Bagging, Random Forest

5.3 A/B Testing

A controlled experiment comparing two versions (A and B) to determine which performs better.

  • Define hypothesis: H₀: No difference between A and B
  • Randomly assign users to groups A and B
  • Collect data and run a statistical test (Z-test, T-test)
  • Calculate p-value and decide based on α = 0.05
  • ⚠️ Avoid peeking at results before reaching required sample size

5.4 Time Series Analysis

  • Trend: Long-term direction of the data
  • Seasonality: Periodic patterns repeating at fixed intervals
  • Cyclical: Patterns over irregular intervals (economic cycles)
  • Stationarity: Mean and variance are constant over time (required for many models)
  • Models: AR, MA, ARMA, ARIMA, SARIMA, Prophet

5.5 PCA — Dimensionality Reduction

Transforms correlated variables into uncorrelated principal components, maximizing explained variance.

  • PC1 explains the most variance, PC2 the next most, etc.
  • Used for visualization (2D/3D), noise reduction, feature engineering

5.6 Bayesian Statistics

Posterior ∝ Likelihood × Prior
  • Prior: Belief about parameter before seeing data
  • Likelihood: How likely the data is given the parameter
  • Posterior: Updated belief after seeing data
  • Credible Interval: Bayesian equivalent of confidence interval
Advanced Interview Questions
Q What is p-hacking and how can it be avoided? +
A P-hacking (data dredging) is running multiple tests and only reporting those with p<0.05, inflating false discovery rates. Avoidance: pre-register hypotheses, use Bonferroni correction (α/n), use FDR control (Benjamini-Hochberg), report effect sizes alongside p-values, use train/test splits in ML contexts.
Q Explain multicollinearity: how to detect and fix it? +
A Multicollinearity occurs when predictor variables are highly correlated, making it hard to isolate individual effects. Detection: VIF (>5 concerning, >10 severe), correlation matrix. Fixes: remove correlated predictors, use PCA for orthogonal components, apply Ridge Regression (L2), or combine correlated variables.
Q What is the difference between L1 and L2 regularization? +
A L1 (Lasso): adds λΣ|βᵢ| to loss; drives some coefficients to exactly 0 — performs feature selection. L2 (Ridge): adds λΣβᵢ² to loss; shrinks all coefficients toward zero but rarely to 0; handles multicollinearity well. ElasticNet combines both. Key difference: L1 does feature selection; L2 does not.
Q What is the bias-variance tradeoff? +
A Bias: error from wrong assumptions → underfitting. Variance: error from sensitivity to training data fluctuations → overfitting. Total Error = Bias² + Variance + Irreducible Noise. As model complexity increases, bias decreases but variance increases. Cross-validation helps find the optimal balance.
Q Explain AUC-ROC and when to use it. +
A AUC-ROC measures a classifier's ability to discriminate between classes across all thresholds. ROC plots TPR vs FPR. AUC=0.5 is random; AUC=1.0 is perfect. Use when: classes are imbalanced, decision threshold isn't fixed, or you need a threshold-independent metric. Prefer Precision-Recall curves when the positive class is very rare.
Q What is heteroscedasticity and why does it matter? +
A Heteroscedasticity means variance of residuals is not constant across levels of independent variables (violates OLS assumption), leading to inefficient estimates and unreliable tests. Detection: Breusch-Pagan test, residual plots. Fixes: log-transform the dependent variable, use Weighted Least Squares (WLS), or robust standard errors (Huber-White).
Q What is the difference between Frequentist and Bayesian statistics? +
A Frequentist: probability = long-run frequency; parameters are fixed; uses p-values and CIs; cannot incorporate prior beliefs. Bayesian: probability = degree of belief; parameters are random variables; uses Bayes' theorem to update priors with data into posteriors; naturally incorporates prior knowledge. Bayesian is more flexible but computationally intensive (MCMC).
Q How do you handle imbalanced datasets in classification? +
A Strategies: Resampling — oversample the minority class (SMOTE) or undersample the majority. Algorithm-level — use class_weight='balanced', adjust decision threshold. Evaluation — use Precision, Recall, F1, AUC-ROC instead of accuracy. Ensemble methods — BalancedRandomForest, EasyEnsemble. Always use stratified cross-validation.
